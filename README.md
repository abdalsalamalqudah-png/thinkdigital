## 🛠️ التقنيات المستخدمة

- **Backend**: Hono Framework على Cloudflare Workers
- **Frontend**: HTML + TailwindCSS + JavaScript
- **Icons**: FontAwesome
- **Slider**: Swiper.js
- **Charts**: Chart.js
- **Database**: Cloudflare D1 (جاهز للربط)
- **Hosting**: Cloudflare Pages
